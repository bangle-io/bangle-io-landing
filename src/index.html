Note taking for the next decade

Bangle.io is a powerful yet simple note taking web app that works on top of your locally saved Markdown files.

Get started

Own your notes

Bangle.io saves all your notes locally in a folder of your choice as plain Markdown files.

Bangle.io provides a clean WYSIWYG interface so you can focus on your notes.

  • Date parsing
  • Emojis
  • Markdown shortcuts
  • Todo lists

Organize notes

Whether you are a folder person or backlink gardener Bangle.io has you covered.

  • Action palette
  • Backlinks
  • Folder view
  • Tags
  • Workspaces
